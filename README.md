# AI for Security and Security for AI — Companion Site

This is a minimal static companion site for the book.

## Recommended hosting

Use GitHub Pages for the public companion site and optionally configure a custom domain.

## Files

- `index.html`: main companion page.
- `notebooks/`: companion notebooks for book labs.

## Suggested printed-book URL

Use a durable page URL, not a raw file URL. For example:

- `https://YOUR-USERNAME.github.io/ai-security-book/`
- `https://ai-security-book.org/`

## Updating

Add new notebooks or errata pages as the book evolves.
