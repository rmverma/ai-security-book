Datasets

  • DiFrauD: deception and fraud domains used for job-scam, few-shot, and lifelong-learning labs.
  • PhiUSIIL Phishing URL Dataset: optional URL-based phishing-detection lab using one unified labeled URL dataset.

Safety note: URL labs should use only dataset-provided strings or features. Do not visit, crawl, or render suspicious URLs.

Using the notebooks

  1. Download the notebook.
  2. Open it in Jupyter, VS Code, or Google Colab.
  3. Run the cells in order. Some notebooks download public datasets at runtime.
  4. Record the metrics and answer the practitioner questions at the end of each lab.
